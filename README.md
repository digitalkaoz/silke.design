# Portfolio Site for Silke.Schoenthal

## Batteries included

* vite (build tool)
* react (rendering library)
* tailwind (ui kit)
* vite-react-ssg (static site generation)
* beasties (extract and inline critical css)
* Typescript (type all the things)
* Eslint (validate correctness)
* prettier (format nicely)
* sharp (convert all images to nicely optimized nextgen versions)

## Developement

install
```bash
$ pnpm i
```

develop
```bash
$ pnpm dev
```

build
```bash
$ pnpm build
```

## Deploy

Everything is deployed through Netlify nicely integrated into Github Workflows.
